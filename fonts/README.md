# Fonts Directory

This directory is for custom fonts used in the LaTeX template.

## Using Custom Fonts

1. Download Inter font from https://rsms.me/inter/
2. Place `.ttf` files here:
   - Inter-Regular.ttf
   - Inter-Bold.ttf
   - Inter-Italic.ttf
   - Inter-BoldItalic.ttf

3. The template will automatically use them

## Fallback

If custom fonts are not present, the template falls back to Arial (system font).
